# Ubike 壞壞
Ubike 車況回報平台

你是否也有一樣的困擾， 常常在租借Ubike時，需要花不少時間在檢查車況。

如果今天有一個車況查詢平台，讓你能迅速了解每台車的車況，將能大大減少你檢查的時間，讓你能在很短的時間內借到好車。
